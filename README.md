# ichimura-lab

研究・練習用コードをプロジェクト単位で管理するリポジトリ。

## 構成

- `projects/`: 練習・研究プロジェクト本体
- `analysis/`: 複数プロジェクトで再利用する解析資産
- `docs/`: 補足ドキュメント
- `scripts/`: ルート共通の補助スクリプト

## 運用ルール

- 新しいテーマは `projects/` 配下に新規プロジェクトとして追加する
- プロジェクト固有のソース、ジョブ、結果、メモは各プロジェクト内に置く
- GPU 性能解析の雛形や共通スクリプトは `analysis/gpu/` に置く
- 実行生成物や profiler 出力は Git で追跡しない
